# Retail Management System (RMS)

## Overview
The RMS is a system designed to help The Warehouse Group manage their day 
to day retail operations across in-store and online channels. The RMS allows 
users to login, process in-store sales, manage inventory, place online orders, 
process payments, generate reports, and maintain the system.

## Organisation
The Warehouse Group, operating under New Zealand's largest listed retail group.

## Prerequisites

* .NET Framework (pre-installed on Windows)
* Visual Studio (or any C# compatible IDE)

## How to Compile and Run

1. Clone the repository: git clone https://https://github.com/erlluriz19-ui/Data-process-and-software-modelling
2. Open the solution file: Retail Management System.sln
3. Press Ctrl + F5 to build and run

## System Features

* User login and logout with role-based access control
* In-store point of sale — scan products, compute totals, process payments, print receipts
* Real-time inventory management with low-stock alerts
* Online order placement, checkout, and cancellation
* Secure payment processing with success and failure handling
* Sales and inventory report generation and export
* IT administrator user management and system maintenance
